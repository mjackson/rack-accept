Class Rack::Accept::Language

  1. lib/rack/accept/language.rb
Parent: Object

Represents an HTTP Accept-Language header according to the HTTP 1.1 specification, and provides several convenience methods for determining acceptable content languages.

See www.w3.org/Protocols/rfc2616/rfc2616-sec14.html for more information.

46:     def matches(language)
47:       values.select {|v|
48:         v == language || v == '*' || (language.match(/^(.+?)-/) && v == $1)
49:       }.sort {|a, b|
50:         # "*" gets least precedence, any others are compared based on length.
